# QBasic-Projects
This repository contains my personal QBasic programs and experiments.  
These projects are not part of any course — they are my own practice and creative coding exercises.

---

## 📘 Contents
- **Basic Programs**
  - Print statements and loops
  - Simple math calculations
- **Fun Projects**
  - Number guessing game
  - Pattern printing
- **Experiments**
  - Using variables and arrays
  - Small interactive programs

---

## 🚀 How to Run
1. Install a QBasic interpreter (such as QB64).  
2. Copy text from file from this repo.  
3. Paste it in the interpreter and run the program.
